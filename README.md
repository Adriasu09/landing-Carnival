# Carnival DOM — La Feria Digital

Landing page for **AZARGame**, a collection of five classic carnival-style games built with vanilla JavaScript, HTML, and CSS.

## Live Demo

[adriasu09.github.io/landing-Carnival](https://adriasu09.github.io/landing-Carnival/)

## Games

- Piedra, Papel o Tijera
- Trileros
- Tiro al Blanco
- Parejas
- Random Lucky

## Tech Stack

- Vanilla JavaScript (ES Modules)
- CSS3 — custom properties, grid, flexbox, glassmorphism
- No frameworks, no build tools

## Project Structure

```
landing-Carnival/
├── index.html
├── main.js          # Game card injection via IntersectionObserver
├── games.js         # Game data and URLs
├── style/
│   └── main.css
└── assets/
    ├── icons/
    └── img/
```

## Run Locally

Just open `index.html` in a browser — no install needed.
